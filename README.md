# Explainable-AI-Framework-for-Early-Prediction-of-Women-s-Health-and-Nutritional-Deficiencies
AI-powered healthcare application for early prediction of PCOS and anemia using explainable machine learning. Integrates SHAP, LIME, RAG, and FLAN-T5 to generate transparent predictions and personalized health recommendations.




##  Features

- Early prediction of PCOS and Anemia
- Explainable AI using SHAP and LIME
- Personalized recommendations using RAG and FLAN-T5
- FastAPI backend for prediction services
- Data preprocessing, feature engineering, model training, and evaluation
- REST API integration

##  Workflow

1. Data Collection
2. Data Cleaning & Preprocessing
3. Feature Engineering
4. Model Training & Evaluation
5. Prediction of PCOS and Anemia
6. Explainability using SHAP & LIME
7. Personalized Recommendations using RAG + FLAN-T5
8. FastAPI Deployment
